# Julia

Julia language support in Vector is provided by the community-maintained [Julia extension](https://github.com/JuliaEditorSupport/vector-julia).
Report issues to: [https://github.com/JuliaEditorSupport/vector-julia/issues](https://github.com/JuliaEditorSupport/vector-julia/issues)

- Tree-sitter: [tree-sitter/tree-sitter-julia](https://github.com/tree-sitter/tree-sitter-julia)
- Language Server: [julia-vscode/LanguageServer.jl](https://github.com/julia-vscode/LanguageServer.jl)

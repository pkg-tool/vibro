# Racket

Racket support is available through the Racket extension.

- Tree-sitter: `tree-sitter-racket`

The [racket-language-server](https://docs.racket-lang.org/racket-language-server/index.html) is not yet supported in Vector, please see [Issue #15789](https://github.com/vector-editor/vector/issues/15789) for more information.
